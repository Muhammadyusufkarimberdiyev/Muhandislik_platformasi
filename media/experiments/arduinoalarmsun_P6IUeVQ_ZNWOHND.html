Quyosh Alarm Soati Loyihasi

Bu loyiha Arduino asosida fotoresistor yordamida yorug'lik darajasini aniqlaydi. Quyosh chiqqanda yorug'lik mayin oshadi va ma'lum chegara qiymatdan oshganda buzzer orqali signal beradi.

Fotoresistor (LDR) yorug'lik intensivligiga qarab qarshiligini o'zgartiradi. Arduino bu o'zgarishlarni o'qiydi va dasturlashtirilgan shart bajarilganda boshqaruv signalini uzatadi.

// Quyosh Alarm Soati - Arduino Kodi
const int ldrPin = A0;
const int buzzerPin = 8;
int lightThreshold = 500;

void setup() {
  pinMode(buzzerPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int lightLevel = analogRead(ldrPin);
  
  if (lightLevel > lightThreshold) {
    digitalWrite(buzzerPin, HIGH);
    delay(500);
    digitalWrite(buzzerPin, LOW);
    delay(500);
  } else {
    digitalWrite(buzzerPin, LOW);
  }
  
  delay(100);
}

Fotoresistor (LDR)

Fotoresistor yorug'lik intensivligiga qarab elektr qarshiligini o'zgartiradi. Qorong'ida qarshilik yuqori, yorug'likda esa past bo'ladi.

Arduino UNO

Arduino UNO - mikrokontroller platasi bo'lib, sensorlardan ma'lumot o'qish va boshqaruv qurilmalariga signal yuborish uchun ishlatiladi.

Buzzer

Aktiv buzzer - elektr signaliga javob sifatida tovush chiqaruvchi qurilma. Loyihada alarm vazifasini bajaradi.

Qarshilik (10kŒ©)

Qarshilik elektr zanjirida tok oqimini cheklash uchun ishlatiladi. Fotoresistor bilan voltage divider zanjirini tashkil etadi.

Breadboard

Breadboard - lehimlashsiz prototipalash uchun mo'ljallangan plata. Komponentlarni oson ulash va sinash imkonini beradi.
